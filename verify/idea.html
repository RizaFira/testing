Artchain TikTok Integration (Demo)

This mock walkthrough demonstrates how Artchain integrates TikTok's Login Kit and Content Posting API for automated video uploads.

Step 1: User Login

User clicks “Connect TikTok”, triggering TikTok OAuth Login via Login Kit. The app requests permissions for user.info.basic, video.upload, and video.publish.

Step 2: Token Received

After login, Artchain receives an access token, stored securely on the backend. This token authorizes future uploads.

{
  "access_token": "mock_tiktok_token_123",
  "scope": "video.upload,video.publish"
}

Step 3: Video Upload

Using TikTok's /video/upload/ and /video/publish/ APIs, the user’s video file is uploaded from Artchain servers.

{
  "status": "success",
  "video_id": "1234567890"
}

Step 4: Confirmation

The app receives a webhook callback confirming that the video was successfully published to TikTok.

{
  "event": "video.publish.success",
  "video_id": "1234567890"
}

Environment: Sandbox testing only.
